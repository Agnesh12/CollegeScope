
# 🎓 CollegeScope

CollegeScope is a comprehensive web platform designed to simplify the college search and selection process for students. It provides detailed insights into colleges, courses, reviews, and admissions data to help students make informed decisions.

## 🚀 Features

- 🔍 Search and filter colleges based on location, ranking, fees, and more
- 🏫 View detailed college profiles with course offerings, fees, and reviews
- ⭐ Add and read student reviews for colleges
- 📊 Compare colleges side-by-side
- 💬 Contact colleges through integrated forms or links

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Spring Boot
- **Database**: MySQL
